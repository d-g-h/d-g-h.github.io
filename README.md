install

```
npm i
```

dev

```
npm run dev
```

inspect

```
npm run dev:inspect
```

test in dev

```
npm test
```

test in prod w/coverage report

```
npm run test:ci
```

prod

```
npm run build
```

lint

```
npm run lint
```

### Deploy

Push to the main branch to trigger `.github/workflows/nextjs.yml`
